Time-Tracker / Data Logging Tool

    You can use the tracker here, or download it below for local use.

    Time-Tracker — 03 Feb 2025 — (20 KB)

    Features -

    1. hours:minutes data logging.

    2. 5 digit integer data logging.

    3. robust display of calculations for;

        cumulative time logged per individual month, all time.

        difference between anticipated value and value logged.

        projected value from the first day logged to the end of the year.

    4. streak counter for each consecutive day all columns have had data logged.

    5. ability to add and remove columns as needed.

    6. automatic saving of all data to localstorage, or an exported file.

    7. simple cli/markdown style chart

    8. open source, github
